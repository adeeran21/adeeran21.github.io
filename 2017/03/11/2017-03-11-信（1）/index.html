信（1）

亲爱的弗林斯特先生：

下午好。在听了您说“没带耳机”这个消息以后，我的心情如五雷轰顶。本想着在《武林外传》——一部充满着欢乐、哲理和侠气的电视剧——营造的欢愉气氛中，给您写信。但如今，我只好静静沉思。

我的先生，您今天有些奇怪。“噢！大概是我熬夜第二天的幻觉吧！”我告诉自己。上午我没能早起，一直到十点多才吃早午饭，我又难受又更难受。其实我现在，说实话，也是有些昏昏沉沉。但是先生，您确实有些行为，让我不理解。就那上午那事说吧！今天你一直叫我斯莫尔·福来阁·诵，我实在不知这称号从何而来。再举个例子吧，今天您帮我提行李，让我着实很感动，但是您总是拍打我的皮托，我拒绝的时候你还不大高兴。这是为什么呢？噢，上帝啊！请指点我吧！

亲爱的先生，此刻的您，想必正在做着某件你享受的事情吧。比如看一些农庄的资讯，比如和另外一些先生一起玩那高贵的桥牌，或者在进行着一些深奥的学习。但我想最可能的，你应该在看那位女士——那位高贵典雅的、我无法直视其光芒的女士——给您写的信件吧！她的文字必定比春风还要温暖，让您的脸上浮现久违的笑容；或者如月亮般圣洁，让一切繁星都为之羞愧。无论您在做什么，这件事情肯定是让您享受的。但是先生，我——伯德·塞弗·福来阁——一个对您的崇敬犹如滔滔江水连绵不绝的小人物，正在毕恭毕敬地给您写信，希望表达我那卑微的请求。

亲爱的先生，我请求您，别在大街上拍打我的皮托好吗？这件事，可是让我感到深深的不安与恐慌呀！先生，我尊敬您，爱您，但您的这种行为 ，着实让我感到无尽的害怕！先生，我不敢表达我的不满，只希望您，为了您的名声——人人敬仰、不可一世、高贵的名声——考虑考虑吧！

感谢先生在百忙之中阅读我的信件。祝您健康与快乐。

伯德·塞弗·福来阁
2017年3月11日
